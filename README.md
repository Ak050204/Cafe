# Friends Cafe Website

## 📖 Overview

Friends Cafe is a responsive website developed to showcase a cafe's menu, services, and contact information. The project focuses on creating a clean and user-friendly interface.

---

## ✨ Features

- Home Page
- About Us
- Menu Section
- Contact Page
- Responsive Design
- Easy Navigation

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

Friends-Cafe/
├── index.html
├── style.css
├── script.js
├── images/

---

## 🚀 Installation

1. Clone the repository
2. Open the project folder
3. Run index.html in your browser
